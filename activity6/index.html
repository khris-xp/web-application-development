Also, it's not hard to write CSS from the beginning and achieve the same layout by less or more efficient markup, css. By using flexbox and BEM, the component markup can be really good and semantic. Secondly, when it goes to complex design, sometimes bootstrap just can't handle it. Eventually you have to write your own css and js lib. Then you will find some conflicts happens… Because bootstrap's style and js lib is so widely covering every aspect of utility css. React library

React. js is an open-source JavaScript library that is used for building user interfaces specifically for single-page applications. It's used for handling the view layer for web and mobile apps. React also allows us to create reusable UI components. React allows developers to create large web applications that can change data, without reloading the page. The main purpose of React is to be fast, scalable, and simple. It works only on user interfaces in the application.
